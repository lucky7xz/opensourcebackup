Arc42 Architektur
OpensourceBackup — Stand B1–B16

1. Einführung und Ziele

Backup Control Plane für 100+ Systeme. Orchestriert Restic, Borg, pgBackRest, Velero. Browser-Dashboard zeigt: "Sind meine Systeme gesichert — und wurde ein Restore getestet?"

6. Laufzeitsicht

API-Chain

HTTP → Recovery → CORS → SecurityHeaders → BodyLimit → Logging → Timeout → Handler

Backup-Flow

Scheduler → BackupJob{pending}
Agent pollt GET /v1/agent/jobs (Bearer)
→ PUT start → restic backup --json → PUT complete {snapshot_id, bytes}
→ Snapshot mit policy.repository_id registriert

Agent-Download

make build-agent-all → dist/agent/v0.1.0/opensourcebackup-agent-{platform}
GET /downloads/agent/{v}/{os} → Binary-Download
